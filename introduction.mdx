## Base URL

All requests go to:
https://toppa.com.ng
## Authentication

Include your Bearer token in every request:
Authorization: Bearer toppa_xxxxxxxxxxxxxxxx
Get your token from **toppa.com.ng/dashboard → Settings → API**

<Warning>
Never expose your token in client-side code. Rotate it immediately if compromised.
</Warning>
